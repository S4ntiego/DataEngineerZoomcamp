# Online Python code environment
jupyter notebook: pip3 install notebook
downgrade jupyter for extensions: pip3 install --upgrade notebook==6.4.12
jupyter autocomplete (only works with older versions of jupyter and python):
- pip3 install jupyter_contrib_nbextensions
- jupyter contrib nbextension install --user
- jupyter nbextension enable hinterland/hinterland
